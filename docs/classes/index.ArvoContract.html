Class ArvoContract<TUri, TType, TAcceptSchema, TEmits>

ArvoContract class represents a contract with defined input and output schemas. It provides methods for validating inputs and outputs based on the contract's specifications. An event handler can be bound to it so the this contract may impose the types on inputs and outputs of it

Type Parameters

  • TUri extends string = string

    The URI of the contract

  • TType extends string = string

    The accept type, defaults to string.

  • TAcceptSchema extends z.ZodTypeAny = z.ZodTypeAny

    The of the data which the contract bound can accept

  • TEmits extends Record<string, z.ZodTypeAny> = Record<string, z.ZodTypeAny>

    The type of records the contract bound handler emits.
