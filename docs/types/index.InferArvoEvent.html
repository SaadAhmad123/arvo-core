A type utility that infers the structure of an ArvoEvent.

Type Parameters

  • T

Template: TData

The type of the event payload

Template: TExtension

Additional extension properties for the event

Template: TType

The literal type string identifying the event

Example

type UserCreatedEvent = InferArvoEvent<ArvoEvent<
  { userId: string; email: string },
  { region: string },
  'user.created'
>>;

// Results in:
// {
//   id: string;
//   source: string;
//   specversion: string;
//   type: 'user.created';
//   subject: string;
//   datacontenttype: string;
//   dataschema: string | null;
//   data: { userId: string; email: string };
//   time: string;
//   region: string;
//   // ... plus ArvoExtension & OpenTelemetryExtension properties
// }
